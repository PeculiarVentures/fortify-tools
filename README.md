# Fortify WEB

### Quick Development Start

 * Clone latest version this repository
 * `npm install`
 * `npm run development`
 
### Quick Production Start

 * Clone latest version this repository
 * `npm install`
 * `npm run production`

### Deploy to gh-pages

 * `npm run deploy`
